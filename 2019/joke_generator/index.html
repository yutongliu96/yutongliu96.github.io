Joke Generator

Written by Yutong LIU with ♥ on December 11, 2019 in Natural Language Processing

The blog is based on the project of joke generator, which aims to produce a certain joke related to the topic words we provide for the network.

Basic idea is inspired by the neural machine translation, which can translate a sequence to another with a different length.

My model is optimized from the previous neural network, which considers topic words as the input language and convert them into a joke style sentence. The attention mechanism is introduced to help the machine understand which topic is more important in the generated sentence.

For training, topic words are selected from Conan talkshow by POS tag predictor from nltk. We selected all the nouns from the sentence, which is better than adjective or other type words during the training. Following results are selected from tons of meaningless generation.

A simplified version is available on my github, the rate of successful generation is about 10%. Considering it is different for human to make a joke by random selected topics, it is hard to evaluate it. If you have opinions or feel interested, please contact with me.
